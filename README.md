# Pts

![image](./guide/assets/pts.png)
Pts is a typescript/es6 library that enables you to compose and express points in spaces.

Get started at [ptsjs.org](https://ptsjs.org)


## Build and test
```
npm start
npm run build
npm test
```

## Generate docs
```
typedoc --readme none --out docs src --name Pts
```
